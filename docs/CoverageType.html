CoverageType¶

The CoverageType module is a critical component of the the by group byCoverageType() in Paletable. This module only contains one class, CoverageType.

class CoverageType.CoverageType¶

Bases: object

The CoverageType class creates an alias called coverage_type that transposes the 12 mc_plan_type_cd columns allowing byCoverageType() to filter by various coverage types. It also plays a role in the backend method for decorating the coverage_type_label column that is included when one runs byCoverageType() on a Paletable object like Enrollment.

Note

PALET users and analysts will not interact with this module directly. It’s purpose is for aliasing and backend decorating that occures when its respective by group is called.
